# Anthropic

Connect Anthropic to Claude, Codex, and ChatGPT through JoAi's hosted MCP app server.

[Open app page](https://joai.ai/en/apps/anthropic) · [Browse all apps](https://joai.ai/apps)

## Works With

- ChatGPT
- Claude
- Codex
- Cursor
- Other MCP clients

## Install

### Claude

```bash
claude plugin marketplace add --scope user JoAiHQ/claude-plugins
claude plugin install --scope user joai-anthropic@joai-claude-plugins
```

### Codex

```bash
codex mcp add joai-anthropic --url https://cortex.joai.ai/mcp/apps/anthropic
```

### Cursor

Copy to local plugins:

```bash
git clone https://github.com/joai-plugins/anthropic.git ~/.cursor/plugins/local/joai-anthropic
```

Or use the [MCP deeplink](cursor://anysphere.cursor-deeplink/mcp/install?name=joai-anthropic&config=eyJtY3BTZXJ2ZXJzIjp7ImpvYWktYW50aHJvcGljIjp7InVybCI6Imh0dHBzOi8vY29ydGV4LmpvYWkuYWkvbWNwL2FwcHMvYW50aHJvcGljIn19fQ==) to install directly.

### ChatGPT

Paste the MCP URL in ChatGPT settings:

```
https://cortex.joai.ai/mcp/apps/anthropic
```
